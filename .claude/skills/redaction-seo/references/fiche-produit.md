# Gabarit — Fiche produit

**Intention** : transactionnelle. Le visiteur est proche de l'achat : il faut le convaincre ET rassurer.
**Longueur cible** : 300 à 600 mots de description (hors caractéristiques).
**Mot-clé type** : [type de support] + [sport] (+ attribut différenciant) — ex. "support mural surf en bois".

## Structure

1. **H1** = nom du produit incluant le mot-clé ("Support mural en chêne pour planche de surf — Modèle X")
2. **Accroche (50–80 mots)** : le bénéfice principal + le contexte d'usage. Projeter le client : sa planche, sa pièce, son problème résolu. Mot-clé principal dans les 2 premières phrases.
3. **H2 — Bénéfices / pour qui** : 2–3 paragraphes ou une liste courte. Adresser les deux intentions (rangement gain de place / exposition déco) ou celle du produit. Citer les types de planches/vélos compatibles AVEC leur vocabulaire (cf. lexique) : "du fish 5'8" au longboard 9'6"", "compatible VTT tout-suspendu jusqu'à 17 kg".
4. **H2 — Caractéristiques techniques** : liste à puces factuelle — matériaux, dimensions, charge max, contenu du colis, visserie, compatibilité murs (placo, brique, béton).
5. **H2 — Installation / entretien** : 2–4 phrases rassurantes (temps de pose, outils, notice). C'est ici qu'on répond à "et sur du placo ?".
6. **Réassurance** : fabrication, garantie, livraison, retours — selon les éléments réels du site (ne rien inventer).
7. **Liens internes** : catégorie parente + 1–2 produits complémentaires ou article de blog ("Comment ranger son quiver").

## Spécifique fiche produit

- **Ne jamais inventer une caractéristique** (charge max, matériau, dimension) : si l'information manque, poser la question ou laisser un placeholder [À COMPLÉTER : charge max].
- Vérifier la **cohérence charge max / poids de l'équipement** (cf. lexique, tableaux de poids).
- Chaque déclinaison/variante a un texte unique (pas de duplicate).
- Title type : "Support mural surf en chêne | [Marque]" — Meta description type : bénéfice + matériau + incitation ("Exposez votre planche comme une œuvre. Chêne massif, pose en 10 min. Livraison rapide.").
- Alt images : produit + équipement + contexte ("support mural noir avec snowboard freestyle dans un salon").
