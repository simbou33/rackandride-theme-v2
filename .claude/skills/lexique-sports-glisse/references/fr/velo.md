# Vélo — Lexique technique pour la rédaction

## Anatomie d'un vélo

- **Cadre (frame)** : avec **tube horizontal (top tube)**, **tube de selle**, **tube diagonal**, **douille de direction**, **bases** et **haubans** (le triangle arrière). Matériaux : aluminium, **carbone** (fragile aux points de serrage — argument clé pour le type de support !), acier, titane
- **Fourche** : rigide ou **suspendue** (VTT)
- **Roues** : jante, rayons, moyeu, **pneus** (ou boyaux / **tubeless**). Diamètres : 26", 27,5", **29"** (VTT), **700c** (route, gravel), 20" (BMX, pliant)
- **Transmission** : **dérailleur** avant/arrière, **cassette**, **plateaux**, chaîne, **pédalier**, manivelles
- **Cockpit** : **cintre** (guidon ; cintre route = drop bar), **potence**, leviers, **cocottes** (route)
- **Selle** et **tige de selle** (télescopique en VTT moderne)
- **Freins** : à disque (hydrauliques) ou sur jante (patins)
- **Pédales** : plates ou **automatiques** (avec cales)
- **Batterie et moteur** : pour les **VAE** (vélos à assistance électrique) — poids nettement supérieur, point crucial pour la charge d'un support

## Types de vélos (poids = donnée n°1 pour un support mural)

| Type | Poids | Particularités rangement |
|---|---|---|
| **Vélo de route** | 7–9 kg | Léger, cintre drop bar, souvent en carbone |
| **Gravel** | 8–11 kg | Pneus plus larges, très tendance (mot-clé fort) |
| **VTT semi-rigide (hardtail)** | 11–14 kg | Fourche suspendue, guidon large |
| **VTT tout-suspendu (enduro, DH)** | 13–17 kg | Encombrant, guidon très large (780–800 mm) |
| **VAE / VTT électrique (VTTAE)** | 18–28 kg | Trop lourd pour beaucoup de supports — vérifier la charge max ! |
| **Vélo urbain / hollandais** | 12–18 kg | Garde-boue, porte-bagages |
| **Vélo pliant** | 10–14 kg | Se range autrement |
| **BMX** | 10–12 kg | Roues 20", culture proche du skate |
| **Fixie / single speed** | 7–9 kg | Esthétique épurée, très exposé en déco intérieure |
| **Vélo vintage / course acier** | 9–12 kg | Pièce de collection murale |

## Modes de fixation murale (vocabulaire précis)

- **Support horizontal (par le cadre)** : le vélo est parallèle au mur, posé sur le tube horizontal — le plus déco
- **Support vertical (par la roue avant)** : le vélo est perpendiculaire/suspendu par la roue — gain de place maximal en garage
- **Crochet vélo** : la solution simple par la jante
- **Râtelier / rack multi-vélos** : pour la famille
- Points d'attention à mentionner : largeur du **cintre** (distance au mur), **garde-boue** et câbles, cadre **carbone** (ne jamais pincer, préférer l'appui), **VAE** (charge max, retirer la batterie), pneus qui marquent le mur

## Vocabulaire du pratiquant

- Cycliste, **vététiste**, **rouleur**, **gravelleux** (familier), commuter (vélotaf)
- **Vélotaf** : aller au travail à vélo — mot-clé urbain en or
- **Sortie, boucle, dénivelé (D+), segment, home trainer** : la pratique route
- **Single (track), enduro, DH (descente), cross-country (XC), bikepark** : la pratique VTT
- **Bikepacking** : voyage à vélo léger (gravel)
- **N+1** : la "règle" du nombre idéal de vélos (nombre actuel + 1) — clin d'œil communautaire parfait pour vendre un deuxième support

## Pièges à éviter

- **La charge max** : un texte crédible distingue toujours vélo route (8 kg) et VAE (25 kg). C'est LE critère d'achat.
- Ne pas confondre **cintre** (vélo) et "guidon" est acceptable, mais "cintre" crédibilise sur la route/gravel.
- Pas de "trucks", "dérives", "fixations" (au sens snowboard) sur un vélo.
- Roues : "pouces" pour VTT/BMX, "700c" pour la route — ne pas mélanger.
