# Skateboard — Lexique technique pour la rédaction

## Anatomie d'un skateboard

- **Deck (plateau)** : la planche en bois, généralement 7 à 9 plis d'érable canadien (**érable 7 plis** est un marqueur de crédibilité). Largeur en pouces : 7.75", 8.0", 8.25", 8.5"...
- **Nose** : l'avant (légèrement plus long et relevé) ; **Tail** : l'arrière
- **Concave** : la courbure transversale du deck
- **Kicks (nose kick / tail kick)** : les parties relevées aux extrémités
- **Grip (griptape)** : la bande antidérapante collée sur le dessus
- **Trucks** : les essieux métalliques qui relient deck et roues. Composés du **baseplate** (platine vissée au deck), du **hanger** (axe), du **kingpin** (vis centrale) et des **gommes (bushings)**
- **Roues (wheels)** : en uréthane ; diamètre en mm (50–60 mm street, 60–75 mm+ cruising/longboard) ; dureté en **duromètre** (99A dures pour le street, 78A tendres pour le cruising)
- **Roulements (bearings)** : norme **ABEC**
- **Visserie (hardware)** : les 8 vis qui fixent les trucks
- **Pads / risers** : cales entre deck et trucks
- **Ply** : un pli de bois ; **pop** : la nervosité du deck (un deck qui perd son pop est "mort")

## Types de planches

| Type | Longueur | Usage |
|---|---|---|
| **Skateboard street / classique** | ~31–32" (80 cm) | Tricks, street, skatepark |
| **Cruiser** | 25–32" | Déplacement urbain, roues tendres |
| **Longboard skate** | 33–60" (85–150 cm) | Descente (downhill), carving, dancing, freeride |
| **Mini-cruiser (type penny)** | 22–27" | Plastique, transport |
| **Old school / shaped** | variable | Formes rétro, pool — souvent des pièces de collection exposées |
| **Surfskate** | 30–34" | Truck avant pivotant, simule le surf — pont parfait entre les deux univers |

## Disciplines et vocabulaire du pratiquant

- **Street, park, bowl, rampe, vert** : les terrains de pratique
- **Tricks** : ollie, kickflip, heelflip, grind, slide, manual... (à citer avec parcimonie)
- **Setup / board complète (complete)** : la planche montée ; beaucoup de skateurs montent leur board eux-mêmes
- **Skateshop** : la boutique spécialisée
- Le skateur garde souvent ses **vieux decks** : decks collectors, séries limitées, artworks d'artistes — énorme angle SEO "**exposer un deck au mur**" / "deck en déco"
- Pratiquants : skateur/skateuse, rideur/rideuse

## Spécificités pour le support mural

- Deux logiques de rangement : **horizontal** (board complète, prête à rider) et **vertical / à plat contre le mur** (deck seul exposé comme un tableau)
- Poids d'une board complète : 3–4 kg ; un longboard : 4–6 kg
- L'**artwork** (la sérigraphie sous le deck) est la face que l'on expose : un support discret/invisible est un argument fort
- Le grip ne doit pas frotter le mur ; les roues peuvent tacher → supports avec contact sur les trucks ou le deck

## Mots-clés SEO naturels

support mural skateboard, accroche skate mural, rack skate, support deck skateboard, exposer un deck au mur, support mural longboard, fixation skateboard mur, ranger son skate, présentoir skateboard, déco skateboard chambre, support skate invisible / discret, porte skateboard.

## Pièges à éviter

- **Longboard** seul est ambigu : préciser "longboard skate" ou "longboard de surf" selon le contexte.
- Ne pas parler de "dérives", "fixations", "guidon" sur un skate.
- "Planche à roulettes" : compréhensible mais daté — l'utiliser éventuellement une fois pour le SEO grand public, jamais comme terme principal.
